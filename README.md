# 🚀 HireAI — AI-Powered Career Platform

HireAI is a full-stack AI-powered career platform that helps users optimize resumes, prepare for interviews, match with jobs, and track applications using modern AI workflows.

Built with modern production-grade technologies including Next.js App Router, PostgreSQL, Prisma ORM, and Google Gemini AI.

# ⚡ Getting Started

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/HireAI.git
cd HireAI
```

---

## 2️⃣ Install Dependencies

```bash
npm install
```

---

## 3️⃣ Setup Environment Variables

Create `.env.local`

```env
DATABASE_URL=
AUTH_SECRET=
GITHUB_ID=
GITHUB_SECRET=
UPLOADTHING_SECRET=
UPLOADTHING_APP_ID=
GEMINI_API_KEY=
```

---

## 4️⃣ Prisma Setup

```bash
npx prisma db push
npx prisma generate
```

---

## 5️⃣ Start Development Server

```bash
npm run dev
```
